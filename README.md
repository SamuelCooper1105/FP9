This project is a Telematics System for a pre-OBD vehicle.
The Raspberry pi 3 B+ used for this project will read real-time data from the 160 baud ALDL port, decode the key sensors, and then display them on a live web dashboard accessible from a phone or laptop. 
Data will be logged for trip summaries and derived metrics like 
- Engine load 
- Coolant warm up time
- Idle stability
- throttle rate
- battery health
- trip summary
- etc

This was a final project for CSE 525 at the university of louisville. Feel free to reuse the code here, I will be changing some small things to hopefully make this much faster as the current code base is rather slow, due to the nature of the serial port translations. I hope to learn more about this and make the project run much smoother. In addition I would like to create a better Web interface as the one currently in this project was generated by AI and does not really do a good job of displaying the information. Refactoring the code will likely be a major part of this but the code works as is right now but will most likely require some changes depending on your vehicle.
